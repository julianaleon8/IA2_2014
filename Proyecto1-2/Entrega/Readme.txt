En la carpeta se encuentran tres directorios

* Data: Contiene todos los datos utilizados en los experimentos
	que aparecen en las gráficas del informe.

* circulo: Contiene las clases de C++ que se utilizan para 
	la clasificación de los puntos en el plano. También 
	en lenguaje Python el archivo generador de casos.
	 - Para compilar: make
	 - Para ejecutar: ./main archivoEntrenaiento nroCasosEntrenamiento 
				NroInput NroOutput archivoTest nroCasosTest 
					NroNeuronasIntermedias

* iris: Contien las clases de C++ que se utilizan para la
	clasificación de los datas de Iris Data Set, tanto 
	la primera parte como la segunda. También se puede
	encontrar con la carpeta de gráficos más relevantes
	del error cuadrado medio de la primera parte.
	 - Para compilar: make
	 - Para ejecutar: ./main archivoEntrenaiento nroCasosEntrenamiento 
				NroInput NroOutput archivoTest nroCasosTest 
					NroNeuronasIntermedias
